🔌 OBS Setup

Setup Steps:

1. Install OBS Virtual Camera plugin
→ obsproject.com → Browse Plugins → VirtualCam

2. In OBS: Start virtual camera
→ Tools → Start Virtual Camera

3. Enable OBS WebSocket
→ Tools → WebSocket Server Settings → Enable
